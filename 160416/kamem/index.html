HTML勉強

HTMLとはハイパーテキストマークアップランゲージの略です。

ブロック要素とインライン要素の違い

ブロック要素

改行するタグ、主に（h1, ul, ol, dl, li, dt, dd, p）

インライン要素

改行しないタグ主に（span, a, em, strong ,small）

タグの説明

  • 見出し : h1 〜 h6
  • 段落 : p
  • リスト : ul, li
  • 番号付きリスト : ol, li
  • 定義付きリスト : dl, dt, dd
  • まえがき : header
  • 本文 : main
  • あとがき : footer
  • 文章の範囲 : section
  • 意味がないがグループかに役立つ: div, span
  • 強調: em, strong
  • 弱める（注釈などに使うことが多い）: small
  1. 見出し : h1 〜 h6
  2. 段落 : p
  3. リスト : ul, li

CSS勉強

CSSとはカスケーディングスタイルシートの略です。

CSSの説明

  • 外側の空き : margin
  • 内側の空き : padding

idとクラスについて

  • idの書き方（#）: #header
  • classの書き方（.）: .copyright
